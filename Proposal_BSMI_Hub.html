1. EXECUTIVE SUMMARY & LATAR BELAKANG

Bulan Sabit Merah Indonesia (BSMI) merupakan organisasi kemanusiaan terdepan yang memegang peranan vital dalam tanggap bencana, pelayanan kesehatan gratis, serta pemberdayaan sosial masyarakat. Seiring dengan meluasnya jaringan cabang, bertambahnya relawan, serta semakin kompleksnya program aksi di lapangan, kebutuhan akan efisiensi pengelolaan data menjadi sangat mendesak.

Pengelolaan administrasi yang masih mengandalkan pencatatan manual atau spreadsheet terpisah kerap memicu beberapa kendala operasional yang signifikan:

  • Risiko Duplikasi & Data Tercecer: Rekapitulasi relawan dan donasi lambat diproses saat kondisi darurat bencana.
  • Transparansi Stok Logistik: Kesulitan memantau stok bantuan di gudang secara real-time.
  • Pelaporan Berjenjang: Membutuhkan waktu lama untuk menyusun laporan keuangan dan aksi lapangan dari cabang ke pusat.

BSMI Hub hadir sebagai solusi sistem terintegrasi berbasis web untuk mendigitalisasi seluruh rantai operasional BSMI — mulai dari pendataan anggota, manajemen tanggap bencana, kontrol logistik, hingga transparansi kas keuangan secara real-time.

4. SPESIFIKASI ARSITEKTUR & TEKNOLOGI

Aplikasi ini dibangun menggunakan Tech-Stack enterprise terkini untuk menjamin stabilitas, kecepatan tinggi, dan kemudahan perawatan jangka panjang:

BE

Laravel 12

Framework PHP enterprise standard dengan tingkat keamanan dan performa tinggi.

FE

React.js & Inertia.js

Menyajikan pengalaman UI ultra-cepat tanpa loading antar halaman.

UI

Tailwind CSS

Interface modern, bersih, dan 100% responsif di smartphone & tablet.

DB

MySQL / MariaDB

Database relasional handal dengan high-availability & performa optimal.

Arsitektur Keamanan Sistem

Keamanan menjadi prioritas utama dalam pengembangan BSMI Hub. Berikut adalah lapisan keamanan yang diterapkan:

  • Enkripsi Data (AES-256): Seluruh data sensitif (password, informasi pribadi anggota) dienkripsi menggunakan standar industri.
  • Role-Based Access Control (RBAC): Setiap pengguna hanya dapat mengakses fitur sesuai dengan peran dan hak akses yang ditetapkan.
  • Proteksi CSRF & XSS: Sistem dilengkapi dengan perlindungan terhadap serangan Cross-Site Request Forgery dan Cross-Site Scripting.
  • Rate Limiting & Throttling: Pencegahan terhadap serangan brute-force login dan DDoS.
  • Audit Log: Pencatatan seluruh aktivitas pengguna untuk keperluan investigasi dan akuntabilitas.

Infrastruktur Hosting & Deployment

Sistem di-hosting menggunakan infrastruktur cloud yang menjamin ketersediaan tinggi (high availability) dengan spesifikasi:

  • Web Server: LiteSpeed / Apache dengan SSL Certificate (HTTPS) untuk koneksi aman.
  • Domain Resmi: Menggunakan domain organisasi (bsmi-banten.org) dengan sertifikat SSL aktif.
  • Backup Otomatis: Sistem backup harian yang menjamin pemulihan data jika terjadi insiden.
  • Uptime Guarantee: Garansi server aktif 99.9% sepanjang tahun.

7. NILAI TAMBAH & KEUNTUNGAN STRATEGIS

  • Pemeliharaan Mudah: Menggunakan teknologi open-source berstandar global yang didukung oleh komunitas pengembang raksasa, sehingga sistem tidak mudah outdated (usang).
  • Siap Di-Skala (Scalable): Struktur database dan arsitektur aplikasi telah dipersiapkan jika ke depannya BSMI ingin mengembangkan sistem ini menjadi Aplikasi Android/iOS secara native.
  • Pengambilan Keputusan Cepat: Pimpinan cabang/pusat dapat langsung melihat kondisi riil keuangan, stok logistik, dan data anggota tanpa harus menunggu staf membuat rekapan Excel di akhir bulan.
  • Keamanan Berlapis: Sistem dilengkapi dengan enkripsi data, hak akses berjenjang (Role-Based Access Control), dan proteksi terhadap serangan siber umum (SQL Injection, XSS, CSRF).
  • Hemat Biaya Jangka Panjang: Mengurangi kebutuhan akan banyaknya staf administrasi dan meminimalisir pembelian perangkat lunak lisensi berbayar.

📋 GARANSI & LAYANAN TAMBAHAN

  • Free Maintenance & Bug Fixing selama 3 bulan pertama pasca Handover.
  • Backup Data Rutin dengan sistem redundansi terproteksi.
  • Support Teknis Prioritas via WhatsApp/Email selama masa garansi.
  • Update Keamanan patch & update framework selama masa garansi.
